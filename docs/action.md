## `u4ic-test action test`

Update a connector action.

```
USAGE
  $ u4ic-test action test [-d <value>] [-h]

FLAGS
  -d, --directory=<value>  [default: ./] Root directory of the repository.
  -h, --help               Show CLI help.

DESCRIPTION
  Update a connector action.
```
